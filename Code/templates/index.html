SCRUM ACADEMY

Nossa plataforma abrange o módulo de treinamento relacionado ao método Scrum, um método ágil responsável pelo desenvolvimento de projetos de maneira flexível e eficiente.

Nesta área, você poderá conferir os conteúdos relacionados ao framework Scrum, em uma interface simples e intuitiva.

Os conteúdos são divididos por capítulos incluindo diferentes tópicos, e ao final de cada um, serão disponibilizados testes para comprovar a fixação dos conceitos teóricos apresentados nos diferentes capítulos da plataforma.
